RAINDROP GAME

Description:
In this game raindrops will drop from the top of the screen in increments and fall to the bottom where a "catcher" controlled by
the user will catch attempt to catch them. 

Objective: 
The objective of this game is for the user controlled catcher to catch as many raindrops that fall as they can and to attempt to 
catch the raindrops before they fall to the bottom. The main objective is to see how long that the player can last. 

Losing:
The player will lose if they fail to catch the raindrops a certain amount of x times. This will result in the game ending and the
screen presenting the player will a losing screen.
